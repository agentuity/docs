---
title: Agentuity
---

Agentuity is a cloud platform designed specifically for building, deploying, and scaling autonomous AI agents. It provides the infrastructure and tools necessary to manage agents built with any framework, such as CrewAI, LangChain, or custom code.

With Agentuity, you can:
- Deploy agents with a single click
- Monitor real-time performance and analytics
- Scale agents effortlessly
- Connect agents to various channels (chat, webhooks, email, SMS, voice)
- Integrate with any AI framework

## Getting Started

import { Step, Steps } from 'fumadocs-ui/components/steps';

<Steps>
<Step>
### Create an Account
[Create an Agentuity account](/docs/CloudPortal) or sign in to the Cloud Portal
</Step>

<Step>
### Install the CLI
```bash
brew tap agentuity/tap && brew install agentuity
```
[More CLI installation options →](/docs/CLI/installation)
</Step>

<Step>
### Login to Agentuity
```bash
agentuity login
```
[CLI authentication details →](/docs/CLI/login)
</Step>

<Step>
### Create Your First Project
```bash
agentuity create project my-project
cd my-project
```
[Project creation guide →](/docs/projects/creating)
</Step>

<Step>
### Deploy Your Project
```bash
agentuity deploy
```
[Deployment documentation →](/docs/projects/deployment)
</Step>
</Steps>
