# API Reference

## Types

### Data

```typescript
interface Data {
  contentType: string;
  base64: string;
  text: string;
  json: Json;
  object<T>(): T;
  binary: Uint8Array;
  buffer: Buffer;
}
```

The `Data` interface provides unified access to content in different formats.

### DataType

```typescript
type DataType =
  | Buffer
  | Uint8Array
  | ArrayBuffer
  | string
  | Json
  | Blob
  | ReadableStream
  | Data;
```

`DataType` represents the different types of data that can be passed to an agent.

### AgentRequest

```typescript
interface AgentRequest {
  get trigger(): TriggerType;
  get data(): Data;
  get metadata(): JsonObject;
  get(key: string, defaultValue?: Json): Json;
}
```

The `AgentRequest` interface provides access to request data.

### AgentResponse

```typescript
interface AgentResponse {
  handoff(
    agent: GetAgentRequestParams, 
    args?: InvocationArguments
  ): Promise<AgentRedirectResponse>;
  
  json(data: Json, metadata?: JsonObject): Promise<AgentResponseData>;
  text(data: string, metadata?: JsonObject): Promise<AgentResponseData>;
  binary(data: DataType, metadata?: JsonObject): Promise<AgentResponseData>;
  // ... other methods
}
```

The `AgentResponse` interface provides methods to create responses.

### AgentContext

```typescript
interface AgentContext {
  logger: Logger;
  kv: KeyValueAPI;
  vector: VectorAPI;
  getAgent(params: GetAgentRequestParams): Promise<RemoteAgent>;
  // ... other properties
}
```

The `AgentContext` provides access to services and utilities.

## Functions

### AgentHandler

```typescript
type AgentHandler = (
  request: AgentRequest,
  response: AgentResponse,
  context: AgentContext
) => Promise<AgentResponseData>;
```

The `AgentHandler` is the main function type for implementing agents.
