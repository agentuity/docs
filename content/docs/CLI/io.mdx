---
title: I/O Commands
description: Commands for managing inputs and outputs in Agentuity projects
---

The `io` commands allow you to manage inputs (sources) and outputs (destinations) for your Agentuity projects.

## io source

Commands for managing input sources.

### Usage

```bash
agentuity io source [command]
```

### Aliases

`agentuity io src`, `agentuity io in`, `agentuity io input`

## io source create

Creates a new input source.

### Usage

```bash
agentuity io source create [flags]
```

### Aliases

`agentuity io source new`

### Flags

| Flag | Description |
| ---- | ----------- |
| `--dir`, `-d` | The root of the project directory |

### Examples

Create a webhook source:
```bash
$ agentuity io source create
# Interactive prompts will guide you through the creation process
✓ webhook source created: src_123456
```

## io source list

Lists all input sources.

### Usage

```bash
agentuity io source list [flags]
```

### Aliases

`agentuity io source show`, `agentuity io source get`

### Flags

| Flag | Description |
| ---- | ----------- |
| `--dir`, `-d` | The root of the project directory |

### Examples

```bash
$ agentuity io source list
type: webhook
  id: src_123456
  config: {"authorization":{"token":"your-token","type":"bearer"}}
```

## io source delete

Deletes an input source.

### Usage

```bash
agentuity io source delete [id] [flags]
```

### Aliases

`agentuity io source del`, `agentuity io source rm`

### Flags

| Flag | Description |
| ---- | ----------- |
| `--dir`, `-d` | The root of the project directory |

### Examples

```bash
$ agentuity io source delete src_123456
✓ source deleted: src_123456
```
