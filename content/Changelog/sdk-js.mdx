This page documents the release history of the [Agentuity JavaScript SDK](https://github.com/agentuity/sdk-js).

## v0.0.106-next.0

<Callout type="info">
  Released: April 24, 2025
</Callout>

### Changes

- Breaking Change! Refactor the data API to have async methods instead of static properties ([PR #81](https://github.com/agentuity/sdk-js/pull/81))
- Deprecate the /run/:id route
- Various code simplifications and fixes
