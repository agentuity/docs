# Agent Errors

## AGENT-001: Agent Not Found

**Error Message**: Agent not found

**Cause**: The specified agent ID does not exist or has been deleted.

**Solution**: 
- Verify the agent ID is correct
- Check if the agent still exists in the project
- Contact support if you believe this is an error

## AGENT-002: Invalid Agent Configuration

**Error Message**: Invalid agent configuration

**Cause**: The agent configuration contains invalid or missing required fields.

**Solution**: 
- Review the agent configuration
- Ensure all required fields are provided
- Validate the configuration format

## AGENT-003: Agent Execution Error

**Error Message**: Agent execution failed

**Cause**: The agent encountered an error during execution.

**Solution**: 
- Check the agent logs for detailed error information
- Verify the agent has necessary permissions
- Review the agent's code and configuration

## AGENT-004: Agent Limit Exceeded

**Error Message**: Agent limit exceeded

**Cause**: You have reached the maximum number of agents allowed in this project.

**Solution**: 
- Review your current agents
- Delete unused agents
- Upgrade your subscription plan

## AGENT-005: Invalid Agent Type

**Error Message**: Invalid agent type

**Cause**: The specified agent type is not supported or invalid.

**Solution**: 
- Check the list of supported agent types
- Update the agent configuration
- Contact support for available agent types
