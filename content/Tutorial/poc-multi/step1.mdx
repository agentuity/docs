# Create a new project

Run the interactive project generator:
```bash
agentuity project create
```

When prompted, select:
- Project name: your choice (e.g., `hello-world`)
- Agent name: your choice (e.g., `agent-hello`)
- Language/runtime:
  - TypeScript (Bun) or
  - Python (uv)
- LLM framework: OpenAI (you'll use Agentuity's gateway to access LLM services)

This creates a new folder with a ready-to-run default agent.
