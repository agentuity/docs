## Installation

MacOS installation with Brew:

```bash
brew tap agentuity/tap && brew install agentuity
```

For other operating systems, you can download the appropriate version from the [GitHub releases page](https://github.com/agentuity/cli/releases).

## Upgrading

To upgrade to the latest version of the Agentuity CLI using Brew on MacOS:

```bash
brew update && brew upgrade agentuity
```

## Manual Installation

You can also manually download and install the Agentuity CLI from the [GitHub releases page](https://github.com/agentuity/cli/releases).

1. Visit the [Agentuity CLI releases page](https://github.com/agentuity/cli/releases)
2. Download the appropriate version for your operating system
3. Extract the downloaded file
4. Add the executable to your system PATH
