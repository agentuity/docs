# Data and DataResult

The `Data` and `DataResult` classes provide utilities for working with different content types.

## Data

The `Data` class is a container for working with the payload of agent data.

### Properties

#### contentType

```python
@property
def contentType(self) -> str
```

The content type of the data (e.g., "text/plain", "application/json").

#### base64

```python
@property
def base64(self) -> str
```

The base64-encoded string of the data.

#### text

```python
@property
def text(self) -> str
```

The data represented as a string.

#### json

```python
@property
def json(self) -> dict
```

The data parsed as JSON.

#### binary

```python
@property
def binary(self) -> bytes
```

The data as a bytes object.

## DataResult

The `DataResult` class represents the result of a data operation.

### Properties

#### data

```python
@property
def data(self) -> Data
```

The data from the result of the operation.


## Example

```python
async def run(request: AgentRequest, response: AgentResponse, context: AgentContext):
    # Get data from key-value storage
    result = await context.kv.get("users", "user123")
    
    if result.exists:
        # Access data properties
        user_data = result.data.json
        
        # Process user data
        return response.json({
            "user": user_data,
            "contentType": result.data.contentType
        })
    else:
        return response.json({"message": "User not found"})
```
