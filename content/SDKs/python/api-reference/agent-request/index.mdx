# AgentRequest

The `AgentRequest` class encapsulates incoming request data with content type-specific accessor methods.

## Properties

### data

```python
@property
def data(self) -> Data
```

Returns a `Data` object containing the request payload.

### trigger

```python
@property
def trigger(self) -> str
```

Returns the trigger type of the request (e.g., "webhook", "manual", "agent").

### metadata

```python
@property
def metadata(self) -> dict
```

Returns the metadata associated with the request.

## Methods


### get

```python
def get(self, key: str, default: Any = None) -> Any
```

Gets a value from the request metadata.

## Example

```python
async def run(request: AgentRequest, response: AgentResponse, context: AgentContext):
    # Validate the request
    request.validate()
    
    # Access request data
    data = request.data.json
    
    # Get the trigger type
    trigger = request.trigger
    
    # Access metadata
    user_id = request.get("userId")
    
    # Return a response
    return response.json({"message": f"Hello, user {user_id}!"})
```
