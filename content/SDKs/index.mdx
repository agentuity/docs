## Choose Your SDK

Agentuity provides native SDKs for Python and JavaScript/TypeScript, enabling you to build AI agents in your preferred language. Both SDKs offer the same powerful features for creating, deploying, and managing autonomous AI agents.

## When to Use Each SDK

### Choose Python SDK When:
- Building **data-intensive agents** that process large datasets
- Integrating with **ML/AI libraries** like scikit-learn, pandas, or numpy
- Developing **backend services** and APIs
- Working with **data science workflows**
- Your team has **Python expertise**

### Choose JavaScript SDK When:
- Building **web applications** with agent integration
- Creating **real-time user interfaces** for agents
- Developing **Node.js microservices**
- Need **TypeScript support** for better developer experience
- Your team has **JavaScript/web development expertise**

## Quick Start Examples

### Python SDK
```python
from agentuity import Agent, AgentContext

async def my_agent(ctx: AgentContext, req: AgentRequest) -> AgentResponse:
    # Process data with pandas, run ML models, etc.
    result = await process_with_ml_model(req.data)
    return AgentResponse(data=result)
```

### JavaScript SDK
```typescript
import { Agent, AgentContext, AgentRequest, AgentResponse } from '@agentuity/sdk';

export default async function myAgent(
  ctx: AgentContext, 
  req: AgentRequest
): Promise<AgentResponse> {
  // Integrate with web APIs, real-time data, etc.
  const result = await processWebData(req.data);
  return { data: result };
}
```
